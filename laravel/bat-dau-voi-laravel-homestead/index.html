    Bắt đầu với Laravel Homestead

    Hướng dẫn cài đặt và sử dụng homestead.
    02 Oct 2014 in Laravel  1 minute read

    Homestead là một cách để phát triển các ứng dụng Laravel trên local. Laravel Homestead đã đóng gói sẵn Vagrant “box” - nó cung cấp môi trường phát triển tuyệt vời mà không cần cài đặt PHP, web server hay bất kỳ một phần mềm server nào khác trên máy tính. Developers không phải lo lắng về việc không tương thích giữa các server software với hệ điều hành của mình. Vagrant boxes hoàn toàn sẵn có, có thể bỏ đi (disposable), nếu bạn cấu hình sai, bạn hoàn toàn có thể hủy nó đi và tạo lại một box mới.
