    Hướng dẫn cài đặt Composer

    Cài đặt Composer trên máy tính
    24 Sep 2014 in Laravel  8.71 minutes read

    Composer là một công cụ để quản lý sự phụ thuộc trong PHP. Nó cho phép developers khai báo các thư viện phụ thuộc mà dự án của họ cần và nó sẽ cài đặt tự động các thư viện đó vào trong dự án thông qua command line (dòng lệnh).

    System Requirements

    Composer yêu cầu máy tính của bạn phải có PHP phiên bản 5.3.2 trở lên. Bạn có thể kiểm tra phiên bản PHP trên máy bằng lệnh php -v hoặc php –version như sau: 

    $ php -v
PHP 5.5.14-2+deb.sury.org~precise+1 (cli) (built: Jul  2 2014 11:56:23) 
Copyright (c) 1997-2014 The PHP Group
Zend Engine v2.5.0, Copyright (c) 1998-2014 Zend Technologies
    with Zend OPcache v7.0.4-dev, Copyright (c) 1999-2014, by Zend Technologies

    Nếu kết quả của bạn không tương tự như trên, bạn cần cài đặt PHP trên máy hoặc nâng cấp phiên bản nếu phiên bản của bạn nhỏ hơn 5.3.2.

    Nếu bạn dùng Windows và bạn gặp phải lỗi:

    C:\>php --version
'php' is not recognized as an internal or external command,
operable program or batch file.

    Khi đó, bạn cần thiết lập PATH với lệnh set PATH=%PATH%;C:\path\to\php trong đó \path\to\php là đường dẫn đến thư mục chứa file php.exe. Giả sử, bạn sử dụng xampp, khi đó bạn nhập vào dòng lệnh sau:

    C:\>set PATH=%PATH%;C:\xampp\php

    Kiểm tra kết quả:

    C:\>php -v
PHP 5.5.11 (cli) (built: Apr  8 2014 15:07:14)
Copyright (c) 1997-2014 The PHP Group
Zend Engine v2.5.0, Copyright (c) 1998-2014 Zend Technologies

    Cài đặt

    Trong bài viết này, tôi hướng dẫn cả 2 phương thức cài đặt locally và globally. Nhưng tôi khuyến nghị bạn nên tiến hành cài đặt theo phương thức Globally trên Linux và sử dụng Installer trên Windows.

    Cài đặt trên Linux

    Để cài đặt Composer trên Linux (tôi sử dụng Ubuntu), máy tính của bạn cần có sẵn phần mềm curl. Bạn kiểm tra phần mềm đã được cài đặt hay chưa bằng lệnh: 

    $ curl --version
curl 7.22.0 (i686-pc-linux-gnu) libcurl/7.22.0 OpenSSL/1.0.1 zlib/1.2.3.4 libidn/1.23 librtmp/2.3
Protocols: dict file ftp ftps gopher http https imap imaps ldap pop3 pop3s rtmp rtsp smtp smtps telnet tftp 
Features: GSS-Negotiate IDN IPv6 Largefile NTLM NTLM_WB SSL libz TLS-SRP

    Nếu kết quả hiển thị phiên bản của curl như trên thì máy bạn đã được cài đặt. Nếu không, bạn dễ dàng cài đặt curl bằng lệnh sau: 

    $ sudo apt-get install curl

    Locally

    Với phương thức này bạn sẽ cài đặt composer cho từng dự án của bạn. Bạn di chuyển đến thư mục chứa project, nhập vào dòng lệnh sau vào Terminal:

    $ curl -sS https://getcomposer.org/installer | php

    Nếu lệnh trên bị lỗi, bạn có thể thay thế bằng lệnh: 

    $ php -r "readfile('https://getcomposer.org/installer');" | php

    Máy tính sẽ kiểm tra cấu hình PHP và sau đó sẽ download composer.phar về thư mục project của bạn.

    Bạn cũng có thể cài đặt Composer vào một thư mục cụ thể bằng cách sử dụng option –install-dir và cung cấp đường dẫn tới thư mục đích cho nó (có thể là đường dẫn tương đối hoặc tuyệt đối đều được):

    $ curl -sS https://getcomposer.org/installer | php -- --install-dir=bin

    Sau đó, bạn chỉ cần chạy php composer.phar để chạy Composer.

    Globally

    Bạn có thể đặt file composer.phar ở bất cứ đâu. Và nếu bạn đặt nó trong đường dẫn của bạn thì bạn có thể truy cập nó như là biến toàn cục. Để download composer.phar về máy (đặt bất cứ đâu bạn muốn), bạn chạy các lệnh sau:

    $ curl -sS https://getcomposer.org/installer | php
$ sudo mv composer.phar /usr/local/bin/composer

    Sau đó, bạn chỉ cần chạy composer thay vì phải chạy php composer.phar.

    Cài đặt trên Windows

    Using the Installer

    Đây là cách dễ nhất để bạn cài đặt Composer về máy. Bạn chỉ cần download và chạy file Composer-Setup.exe. Nó sẽ tự động cài đặt phiên bản Composer mới nhất và thiết lập PATH cho bạn. Giờ đây, bạn có thể chạy lệnh composer từ bất cứ thư mục nào bằng command line.

    Manual Installation

    Thay đổi đến thư mục bạn muốn và chạy các lệnh sau để download composer.phar:

    C:\Users\username>cd C:\bin
C:\bin>php -r "readfile('https://getcomposer.org/installer');" | php

    Lưu ý: Nếu câu lệnh trên bị lỗi, bạn thay thế https bằng http hoặc enable php_openssl.dll trong file php.ini.

    Tiếp đến, bạn tạo file composer.bat ngang hàng với file composer.phar bằng lệnh sau: 

    C:\bin>echo @php "%~dp0composer.phar" %*>composer.bat

    Sau khi cài đặt xong, để kiểm tra việc cài đặt composer, bạn đóng cửa sổ Ternimal hiện tại lại, mở một cửa sổ terminal mới: 

    C:\Users\username>composer --version
Composer version b33da336ecb7f1c9b15a57285e5a9f09cf5f3dd2 2014-09-30 15:28:01

    Với kết quả hiển thị như trên, bạn đã cài đặt thành công Composer trên máy tính của mình.
