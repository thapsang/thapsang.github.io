    Xóa cache views trong laravel 4 với Artisan command

    Xóa cache views bằng dòng lệnh.
    03 Oct 2014 in Laravel  3.42 minutes read

    Khi phát triển ứng dụng với laravel 4, có sự thay đổi trên view nhưng khi reload ứng dụng thì bạn không hề thấy có sự thay đổi. Mặc dù đã xóa cache với lệnh php artisan cache:clear nhưng vẫn không khả quan hơn. Lý do là ứng dụng của chúng ta còn lưu lại cache của views trong thư mục app/storage/views. Bạn có thể truy cập vào thư mục app/storage/views và xóa hết các files trong thư mục này đi. Với cách làm thủ công này chắc chắn sẽ khiến bạn phát rồ lên. Trong bài này, tôi sẽ chỉ cho bạn cách xóa views cache dễ như ăn kẹo vậy :).

    Chúng ta sẽ chỉ cần chạy lệnh php artisan views:clear là giải quyết được vấn đề với views cache. Để làm được việc đó, chúng ta cần làm những việc sau:

    • Building A Command
    • Registering Command
    • Run Command

    Building A Command

    Laravel cho phép chúng ta tự tạo ra các lệnh của riêng mình. Để tạo một lệnh mới, bạn mở trình Terminal lên, di chuyển đến thư mục chứa project và nhập vào dòng lệnh sau:

    php artisan command:make ViewsCommand

    Với lệnh trên, file ViewsCommand.php sẽ được generate vào thư mục app/commands. Tiếp đến, bạn copy đoạn code dưới đây vào file ViewsCommand.php. Bạn có thể xem thêm chi tiết về cách viết Artisan Command tại Artisan Development :

    Registering Command

    Sau khi ViewsCommand được hoàn tất, bạn cần đăng ký nó với Artisan để có thể sử dụng nó. Bạn mở file app/start/artisan.php và sử dụng method Artisan::add để đăng ký như sau:

    Artisan::add(new ViewsCommand);

    Run Command

    Bây giờ, bạn đã có thể xóa cache views một cách dễ dàng bằng cách chạy lệnh php artisan views:clear. Kết quả như sau:

    $ php artisan views:clear
Views deleted from cache

    Hy vọng có thể giúp các bạn giải quyết được vấn đề của mình. Have fun! ;)
