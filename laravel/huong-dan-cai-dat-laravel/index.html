    Hướng dẫn cài đặt Laravel 4

    Các cách cài đặt Laravel trên local.
    Ta Ly 27 Sep 2014 in Laravel  10.04 minutes read

    Có nhiều cách để cài đặt Laravel. Trong bài viết này, tôi sẽ trình bày các cách cài đặt Laravel trên Linux và trên Windows từ mọi thứ mà Laravel yêu cầu. Bạn sẽ biết cài đặt PHP, MySQL, MCrypt, Composer và mọi thứ để ứng dụng của bạn có thể chạy :).

    Nội dung

    • System Requirements

      • Using LAMPP or XAMPP

    System Requirements

    Để chạy được Laravel, chúng ta cần:

    1. PHP >= 5.4 (phiên bản 5.4 hoặc mới hơn)
    2. MCrypt PHP Extension
    3. Composer
    4. Laravel
    5. Openssl phải được enabled trong file php.ini

    Install PHP

    Trước hết, bạn kiểm tra PHP đã được cài đặt trên máy hay chưa với lệnh:

    $ php -v

    Nếu kết quả hiển thị như dưới đây, tức là bạn đã cài đặt thành công PHP trên máy tính và bạn có thể bỏ qua phần cài đặt PHP này.

    PHP 5.5.11 (cli) (built: Apr  8 2014 15:07:14)
Copyright (c) 1997-2014 The PHP Group
Zend Engine v2.5.0, Copyright (c) 1998-2014 Zend Technologies

    Nếu phiên bản PHP của bạn nhỏ hơn 5.4 hoặc không có phiên bản nào, bạn cần phải cài đặt PHP. Bạn có thể lựa chọn cài đặt đủ bộ AMP (Apache + MySQL + PHP) bằng cách cài đặt LAMPP (cho Linux) hoặc XAMPP (cho Windows). Hoặc bạn có thể cài trực tiếp các thành phần riêng lẻ.

    Using LAMPP or XAMPP

    Nếu bạn ngại phải cài đặt từng thứ rời rạc, và lo ngại không biết mình có cài đặt và cấu hình đúng không, bạn có thể cài đặt XAMPP để có môi trường đã được tích hợp đầy đủ PHP, Apache2, MySQL, Perl với phiên bản mới nhất. Và một điều rất tuyệt nữa đó là MCrypt đã có sẵn trong XAMPP, do đó bạn không còn phải lo lắng về nó nữa.Nếu bạn dùng Windows, tôi khuyến nghị bạn nên cài đặt PHP theo cách này.

    Để cài đặt XAMPP, bạn cần download XAMPP phiên bản phù hợp với máy tính của bạn về. Sauu đó tiến hành cài đặt lên máy tính của bạn.

    Sau khi hoàn tất, XAMPP được cài đặt vào trong thư mục /opt/lampp với Linux, và trong thư mục C:\xampp trên Windows (nếu bạn chọn ổ C khi cài đặt). Bạn chạy lại lệnh php -v để kiểm tra kết quả.

    Note: Trên Windows, nếu bạn đã cài đặt XAMPP mà vẫn bị báo lỗi không có phiên bản, bạn cần thiết lập PATH với lệnh set PATH=%PATH%;C:\path\to\php trong đó \path\to\php là đường dẫn đến thư mục chứa file php.exe. Giả sử, bạn sử dụng xampp, khi đó bạn nhập vào dòng lệnh sau:

    C:\>set PATH=%PATH%;C:\xampp\php

    Sau khi cài đặt XAMPP, bạn cần Enable OpenSSL bằng cách mở file php.ini (tại thư mục C:/xampp/php), tìm: 

    ;extension=php_openssl.dll

    và bỏ dấu ; phía trước, khi đó, trông nó thế này:

    extension=php_openssl.dll

    Okay, bây giờ thì restart Apache.

    From Scratch

    (Sử dụng cho Linux).

    Nhập vào Terminal các lệnh sau:

    $ sudo apt-get update
$ sudo apt-get install php5-common libapache2-mod-php5 php5-cli

    Sau đó, bạn chạy lại lệnh php -v để kiểm tra phiên bản PHP.

    Install Composer

    Laravel sử dụng Composer để quản lý các thư viện phụ thuộc. Bạn tham khảo bài viết hướng dẫn cài đặt Composer để có thể cài đặt Composer. Tôi khuyến nghị bạn nên cài đặt Composer sử dụng cho toàn hệ thống (globally).

    Bạn có thể kiểm tra Composer đã được cài đặt hay chưa bằng cách mở Terminal và chạy lệnh sau:

    $ composer --version
Composer version b33da336ecb7f1c9b15a57285e5a9f09cf5f3dd2 2014-09-30 15:28:01

    Nếu kết quả hiển thị như trên thì chắc chắn bạn đã cài composer thành công trên máy tính của mình.

    Install Laravel

    Bạn có thể cài đặt Laravel bằng 1 trong 2 cách dưới đây:

    Via Composer

    (Khuyến nghị bạn nên sử dụng cách này).

    Nếu bạn dùng XAMPP, bạn cần di chuyển bằng command line đến thư mục webroot ( /opt/lampp/htdocs trên Linux và C:\xampp\htdocs trên Windows). Bạn muốn tạo mới một project có tên test_laravel, bạn nhập vào Terminal (trình command line trên Windows) như sau:

    $ cd /opt/lampp/htdocs
$ composer create-project laravel/laravel test_laravel --prefer-dist

    Trên Windows sẽ thế này:

    C:\>cd xampp\htdocs
C:\xampp\htdocs> composer create-project laravel/laravel test_laravel --prefer-dist

    Composer sẽ tự động tạo project cho bạn, và bạn chỉ cần ngồi đợi thông báo successfull thôi.

    Via Laravel Installer

    Trước tiên, bạn cần download Laravel Installer. Có 2 cách:

    Cách 1: Sử dụng Composer. Nhập vào Terminal lệnh sau:

    > composer global require "laravel/installer=~1.1"
>

    Cách 2: Download Laravel Installer , đổi tên thành laravel. Tiếp đến, copy file laravel sang thư mục /urs/local/bin.

    Tạo project test_laravel bằng câu lệnh:

    laravel new test_laravel

    Sau khi tạo xong project, bạn cần cấp quyền ghi cho web server vào thư mục app/storage. Nếu bạn dùng Linux, bạn di chuyển đến thư mục chứa project và sử dụng lệnh sau:

    $ sudo chmod -R 777 app/storage

    Run your fisrt app

    Bạn đã có project, bạn cần khởi động nó bằng cách sử dụng Artisan - một command line interface của Laravel. Đơn giản là bạn chỉ cần di chuyển đến project:

    cd test_laravel

    và thực thi lệnh: 

    php artisan server

    Bạn sẽ thấy kết quả: 

    Laravel development server started on http://localhost:8000
