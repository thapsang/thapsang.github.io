    Hướng dẫn cài đặt Ionic Framework trên Ubuntu

    21 Jan 2015 and last modified on 09 Mar 2015 in Ionic  3.88 minutes read

    Trong bài viết này, chúng ta sẽ đi vào chi tiết cách thức cài đặt IonicFramework trên Ubuntu (14.04).

    Nội dung

    • Cài đặt Ionic
    • Khởi tạo ứng dụng
    • Chạy ứng dụng
    • Tài liệu tham khảo

    Cài đặt Ionic

    Trước hết, bạn cần cài đặt Apache Cordova phiên bản mới nhất bằng lệnh:

    $ sudo npm install -g cordova

    Sau đó, bạn có thể kiểm tra phiên bản Cordova vừa được cài đặt với lệnh:

    $ cordova -v

    Tiếp đến, bạn cài đặt Ionic bằng lệnh:

    $ sudo npm install -g ionic

    Khởi tạo ứng dụng

    Bây giờ, chúng ta cần tạo mới một ứng dụng Cordova ở bất kỳ đâu bạn muốn trên máy tính. Giả sử, tôi tạo các ứng dụng ionic của mình trong thư mục home/taly/projects, chúng ta di chuyển tới thư mục home/taly/projects. Bạn có thể tạo ứng dụng sử dụng template có sẵn (sidemenu hoặc tabs) của Ionic hoặc tạo ứng dụng trống rỗng (blank). Ví dụ, chúng ta tạo ứng dụng tên Hello world với template sidemenu như sau:

    $ ionic start hello-world sidemenu

    Chạy ứng dụng

    Sau khi tạo xong ứng dụng, bạn di chuyển vào thư mục hello-world:

    $ cd hello-world

    Bạn có thể xem luôn ứng dụng vừa tạo có hình thù, hoạt động ra sao trên trình duyệt web (còn gọi là web view) bằng cách chạy lệnh:

    $ ionic serve

    Đến đây, chúng ta đã có thể phát triển tiếp ứng dụng của mình với giả lập giao diện trên web view tại địa chỉ mặc định là http://localhost:8100.

    Để build và test trên các nền tảng khác nhau (ios, android), chúng ta cần add platform và build trên các nền tảng tương ứng.

    $ ionic platform add ios
$ ionic platform add android

    Bạn chạy thử ứng dụng (tương ứng với từng platform) với lệnh:

    $ ionic build android
$ ionic emulate android
