    Hướng dẫn cài đặt Android SDK trên Ubuntu

    Cài đặt Android SDK Tool để develop các ứng dụng Ionic trên Ubuntu 14.04
    11 Mar 2015 in Ionic  8.06 minutes read

    Khi bạn muốn build ứng dụng trên các nền tảng khác nhau, máy tính của bạn cần phải được cài đặt các Tool tương ứng. Bài viết này đi vào hướng dẫn cài đặt Android SDK Tool trên Ubuntu 14.04 để develop ứng dụng trên nền tảng Android.

    Nội dung

    • Cài đặt Java
    • Cài đặt Ant
    • Cài đặt Android SDK Tool
    • Tài liệu tham khảo

    Cài đặt Java

    Bạn cần đảm bảo rằng máy tính của bạn đã được cài đặt Java. Bạn có thể kiểm tra với lệnh:

    $ java -version

    Nếu máy bạn đã cài đặt Java, bạn có thể bỏ qua bước này.

    Nếu máy tính của bạn chưa cài đặt Java, chúng ta sẽ tiến hành cài đặt. Bạn chạy các lệnh sau trên Terminal:

    $ sudo dpkg --add-architecture i386
$ sudo apt-get update
$ sudo apt-get install libncurses5:i386 libstdc++6:i386 zlib1g:i386

    Tiếp đến là cài đặt Java (ở đây tôi chọn cài đặt oracle java, bạn có thể cài bản khác nếu muốn):

    $ sudo add-apt-repository ppa:webupd8team/java
$ sudo apt-get update
$ sudo apt-get install oracle-java7-installer

    Nếu bạn muốn cài đặt java8, bạn thay thế lệnh cuối bằng lệnh sudo apt-get install oracle-java8-installer

    Sau khi cài đặt xong, bạn có thể kiểm tra version java được cài đặt trên máy. Kết quả trên máy của tôi như sau:

    $ java -version
java version "1.7.0_76"
Java(TM) SE Runtime Environment (build 1.7.0_76-b13)
Java HotSpot(TM) Server VM (build 24.76-b04, mixed mode)

    Bạn cần thiết lập biến môi trường JAVA_HOME. Các bước thực hiện như sau:

    1. Xác định vị trí đã cài đặt Java trên máy của bạn với lệnh: 
    $ which java
    1. Tiếp đến, chúng ta thiết lập biến toàn cục JAVA_HOME, chúng ta sẽ sửa trong file .bashrc:
    $ gedit .bashrc

    Nhập vào phía dưới cùng của file .bashrc những dòng sau:

    JAVA_HOME=/usr/lib/jvm/java-7-oracle
export JAVA_HOME
PATH=$PATH:$JAVA_HOME/bin
export PATH
    1. Bạn đóng cửa sổ Terminal và bật lại, kiểm tra biến JAVA_HOME đã được thiết lập thành công hay chưa bằng lệnh:
    $ echo $JAVA_HOME

    Cài đặt Ant

    Apache Ant là một hệ thống được xây dựng cho Java, được Cordova và Android SDK sử dụng. Để cài đặt Ant, bạn làm theo các bước sau:

    1. Download Ant tại địa chỉ http://ant.apache.org/bindownload.cgi, lấy bản zip có sẵn trên trang đó, ví dụ, apache-ant-1.9.4-bin.zip, và lưu file trên máy tính của bạn.
    2. Giải nén tập tin vừa download. Copy vào nơi bạn muốn. Tôi sẽ copy vào trong thư mục home/taly/development và rename thư mục thành apache-ant.
    3. Tiếp đến, chúng ta cần thêm Ant vào system path. Chúng ta sẽ sửa file .bashrc, thêm vào gần cuối file (trước dòng export PATH) các dòng sau: 
    ANT_HOME=~/development/apache-ant
export ANT_HOME
PATH=$PATH:$ANT_HOME/bin
    1. Kiểm tra Ant đã được cài đặt thành công hay chưa với lệnh:
    $ ant -version

    Cài đặt Android SDK Tool

    Bạn làm theo các bước sau:

    1. Truy cập vào trang developer.android.com/sdk, di chuyển xuống phần SDK Tools Only, và download file android-sdk_r24.0.2-linux.tgz
    2. Giải nén tập tin, rename thành adroid-sdk-linux, copy vào thư mục development (/home/taly/development).
    3. Thêm Adroid SDK vào system path. Bạn thêm các dòng sau vào trước dòng export PATH trong file .bashrc:
    PATH=$PATH:~/development/android-sdk-linux/platform-tools
PATH=$PATH:~/development/android-sdk-linux/tools

    Đảm bảo rằng trong file .bashrc chứa nội dung như sau:

    JAVA_HOME=/usr/lib/jvm/java-7-oracle
export JAVA_HOME
PATH=$PATH:$JAVA_HOME/bin

ANT_HOME=~/development/apache-ant
export ANT_HOME
PATH=$PATH:$ANT_HOME/bin

PATH=$PATH:~/development/android-sdk-linux/platform-tools
PATH=$PATH:~/development/android-sdk-linux/tools

export PATH
    1. Bạn cần phải có phiên bản Android SDK cụ thể. Để làm việc này, chúng ta chạy Android SDK Manager bằng cách di chuyển đến thư mục android-sdk-linux/tools và chạy lệnh android sdk:
    $ cd ~/development/android-sdk-linux/tools
$ android sdk

    Lần đầu tiên bạn khởi chạy Android SDK Manager sẽ có các thiết lập chọn trước. Đó là các khuyến nghị nên được cài đặt. Ngoài ra, bạn hãy chọn mục Android 4.4.2 (API 19) chưa được cài đặt và tiến hành cài đặt chúng. Đây là phiên bản được Cordova 3.6 sử dụng.

    Lưu ý rằng trong tương lai,các phiên bản Cordova thay đổi cũng sẽ yêu cầu Android SDK tương ứng. Khi đó chúng ta mở lại Android SDK Manager và cài đặt các phiên bản API được yêu cầu.
