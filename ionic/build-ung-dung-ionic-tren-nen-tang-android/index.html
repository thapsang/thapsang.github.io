    Build ứng dụng Ionic trên nền tảng Android

    Test ứng dụng với nền tảng Android phát triển trên Ubuntu 14.04.
    10 Mar 2015 in Ionic  5.27 minutes read

    Sau khi chạy được ứng dụng trên web view, chúng ta sẽ muốn chạy thử ứng dụng của mình trên các nền tảng khác nhau. Trong bài viết này, chúng ta sẽ đi vào chi tiết cách thức add platform và test ứng dụng như thế nào.

    Nội dung

    Cài đặt platform tools

    Trước khi tiến hành add platform, chúng ta cần cài đặt Android SDK.

    System Requirements

    Bạn có thể tham khảo yêu cầu về hệ thống để cài đặt Android SDK. Dưới đây là yêu cầu về hệ thống khi phát triển trên Linux:

    • GNOME or KDE desktop
    • GNU C Library (glibc) 2.15 or later
    • 2 GB RAM minimum, 4 GB RAM recommended
    • 400 MB hard disk space
    • At least 1 GB for Android SDK, emulator system images, and caches
    • 1280 x 800 minimum screen resolution
    • Oracle® Java Development Kit (JDK) 7

    Download và cài đặt SDK Tool

    Bạn có thể download SDK Tool tại đây và đảm bảo rằng bạn đã cài đặt Android SDK theo đúng hướng dẫn và thêm các gói SDK Packages để develop. Bạn có thể tham khảo bài viết Hướng dẫn cài đặt Android SDK trên Ubuntu 14.04 của tôi.

    Để Cordova command-line (CLI) hoạt động đúng, chúng ta cần thêm thư mục tools và platform-tools của SDK vào trong PATH. Bạn có thể sử dụng editor để tạo hoặc chỉnh sửa file ~/.bash_profile, thêm dòng sau vào file (phụ thuộc vào nơi chúng ta cài đặt sdk, thư mục của tôi là /home/taly/development):

    export PATH=${PATH}:/home/taly/development/android-sdk-linux/platform-tools:/home/taly/development/android-sdk-linux/tools

    Tiếp đó, trên Terminal, bạn chạy lệnh sau để load lại file này: 

    $ source ~/.bash_profile

    Add platform

    Bạn có thể kiểm tra xem hệ điều hành của bạn hỗ trợ phát triển những platform nào bằng lệnh:

    $ ionic platform list

    hoặc 

    $ cordova platform ls

    Chúng ta add platform android với lệnh: 

    $ ionic platform add android

    hoặc

    $ cordova platform add android

    Khi bạn chạy lệnh platform add, ionic sẽ tạo ra thư mục android bên trong thư mục platforms và toàn bộ mã nguồn của ứng dụng sẽ được tự động copy vào trong thư mục platforms/android/assets/www. Lưu ý: Bạn không nên chỉnh sửa thông tin gì trong thư mục này trừ khi bạn hiểu rõ mình đang làm gì. Bạn chỉ cần chỉnh sửa mã nguồn của dự án trong thư mục www ở thư mục gốc của dự án. Mỗi lần build trên nền tảng nào, ionic sẽ tự động cập nhật cho bạn.

    Test ứng dụng

    Bạn có thể build và chạy thử xem ứng dụng hoạt động tốt hay không:

    $ ionic build android
$ ionic emulate android
