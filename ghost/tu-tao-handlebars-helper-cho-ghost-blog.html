    Tự tạo handlebars helper cho Ghost blog

    Cách tự tạo file helper sử dụng Ghost blog của bạn.
    11 May 2015 in Ghost  4.56 minutes read

    Ghost blog hiện còn khá mới mẻ và vẫn đang tiếp tục phát triển. Do đó, Ghost chưa thể cung cấp được hết các yêu cầu cho các developer. Tuy vậy, các developer có thể tự tạo cho mình các helper phục vụ cho mục đích của mình. Bài viết này sẽ hướng dẫn cách bạn tự tạo helper cho riêng mình.

    Theo các tài liệu tôi đã đọc cho đến thời điểm viết bài, có 2 cách bạn có thể tạo helper;

    1. Thêm file helper riêng của bạn vào trong thư mục core/server/helpers/.
    2. Tạo file helper riêng của bạn trong thư mục gốc của blog và khởi tạo nó trong file config.js

    Bạn nên chọn cách thứ 2 vì file helper của bạn sẽ không bị mất mỗi khi bạn nâng cấp Ghost blog lên phiên bản mới. Sau đây là chi tiết cách thực hiện.

    Registering a Helper

    Để đăng ký một Helper, bạn tạo ra một file helper.js và đặt nó trong thư mục gốc của blog. Bạn có thể sử dụng một ví dụ trên StackOverflow

    helper.js

    var hbs = require('express-hbs');

module.exports = function(){  
    hbs.registerHelper('cond', function (v1, operator, v2, options) {
        switch (operator) {
            case '==':
                return (v1 == v2) ? options.fn(this) : options.inverse(this);
            case '===':
                return (v1 === v2) ? options.fn(this) : options.inverse(this);
            case '<':
                return (v1 < v2) ? options.fn(this) : options.inverse(this);
            case '<=':
                return (v1 <= v2) ? options.fn(this) : options.inverse(this);
               case '>':
                return (v1 > v2) ? options.fn(this) : options.inverse(this);
               case '>=':
                return (v1 >= v2) ? options.fn(this) : options.inverse(this);
            case '&&':
                return (v1 && v2) ? options.fn(this) : options.inverse(this);
            case '||':
                return (v1 || v2) ? options.fn(this) : options.inverse(this);
            default:
                return options.inverse(this);
        }
    });
};

    Initializing your Helper

    Để Ghost có thể sử dụng được Helper của bạn, bạn cần phải thực hiện một module mới trong file config.js. Chúng ta có thể dễ dàng thực hiện việc đó bằng cách thêm require(‘./helpers’)(); vào phía trên của file config.js. Khi đó, file config.js sẽ trông như sau:

    // # Ghost Configuration
// Setup your Ghost install for various environments
// Documentation can be found at http://support.ghost.org/config/

// Include any helper functions
require('./helpers')();

var path = require('path'),  
    config;

    Kết quả

    Bây giờ chúng ta đã có thể sử dụng Helper của riêng mình. Giả sử trong file index.js trong thư mục theme, sau chúng ta thêm đoạn code sau:

    {{#cond 5 '==' 5 }}
    <script>alert("Success!")</script>
{{/cond}}

    Nếu 5 == 5, chúng ta sẽ hiển thị thông báo Success. Như vậy là chúng ta đã có thể tạo được Helper riêng cho mình. Nhớ restart blog nhé.
