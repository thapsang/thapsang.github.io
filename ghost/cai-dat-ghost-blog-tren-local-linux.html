    Cài đặt Ghost blog trên local Linux

    Hướng dẫn cài đặt Ghost blog trên Linux
    06 May 2015 in Ghost  3.03 minutes read

    Nếu bạn muốn tự host Ghost blog trên máy tính Linux, bạn làm theo các bước sau:

    Cài đặt Node

    Ghost yêu cầu sử dụng Node phiên bản 0.10.x. Tại thời điểm viết bài, Team Ghost khuyến nghị các bạn sử dụng Node version 0.10.x và npm version 2.5.0.

    • Bạn có thể sử dụng các gói quản lý các bản phân phối Node trong theo hướng dẫn trong trang Installing Node.js via package manager hoặc bạn có thể tải bản đóng gói .taz.gz tại trang http://nodejs.org/.
    • Sau đó sử dụng lệnh node -v và npm -v để kiểm tra phiên bản node và npm mà bạn mới cài đặt.

    Cài đặt Ghost

    • Bạn lấy phiên bản mới nhất của Ghost từ ghost.org với lệnh: $ curl -L https://ghost.org/zip/ghost-latest.zip -o ghost.zip
    • Giải nén Ghost vào thư mục bạn mong muốn (giả sử vào thư mục /var/www/ghost với lệnh: $ unzip -uo ghost.zip -d /var/www/ghost
    • Di chuyển vào thư mục ghost vừa mới tạo và cài đặt Ghost: $ cd /var/www/ghost && npm install Nếu bạn muốn cài đặt ở chế độ production, bạn có thể dùng lệnh sau: $ cd /var/www/ghost && npm install –production

    Khởi động Ghost

    • Để khởi động Ghost, bạn chạy lệnh: $ npm start (Nếu bạn muốn chạy trong môi trường production, sử dụng lệnh npm start –production).

    • Bây giờ, Ghost đang được chạy mặc định tại địa chỉ 172.0.0.1:2368 (hay localhost:2368).
    • Bạn truy cập vào địa chỉ http://172.0.0.1:2368 trên trình duyệt để thấy giao hiện Ghost mới thiết lập của bạn.
    • Muốn truy cập vào admin, bạn gõ vào trình duyệt địa chỉ http://172.0.0.1:2368/ghost và tạo tài khoản admin của bạn để đăng nhập vào trang admin.
