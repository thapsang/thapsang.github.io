    Tạo trang contact và gửi mail với Nodemailer cho Ghost blog

    Hướng dẫn chi tiết cách tạo trang contact form và thiết lập gửi mail với Nodemailer cho Ghost blog.
    09 May 2015 in Ghost  7.8 minutes read

    Trước khi thực hiện theo nội dung bài này, bạn cần đảm bảo rằng blog của bạn đã được thiết lập có thể gửi mail. Bạn có thể tham khảo bài Thiết lập gửi mail sử dụng Gmail cho Ghost blog để chắc chắn điều này.

    Nội dung

    • Thêm contact form vào blog
      • Tạo trang contact tĩnh
      • Tạo trang contact bằng template page-contact.hbs
    • Thêm trang contact vào navigation menu
    • Cập nhật frontend controller
      • Tham chiếu đến Nodemailer
      • Thêm phương thức xử lý submit form
      • Thêm đường dẫn xử lý form submit
    • Điều hướng đến một trang tĩnh
    • Tài liệu tham khảo

    Thêm contact form vào blog

    Bạn có thể tạo trang contact cho blog bằng cách tạo một trang tĩnh (static page) và tạo ra một template với tên page-contact.hbs trong thư mục theme của bạn. Bạn có thể xem chi tiết tại http://support.ghost.org/add-contact-form-ghost-blog/.

    Tạo trang contact tĩnh

    1. Đăng nhập vào blog của bạn.
    2. Click vào nút New post ở phía trên navigation menu.
    3. Nhập vào tiêu đề của trang.

    4. Chỉnh sửa các thiết lập cho post bằng cách click vào biểu tượng bên cạnh nút Save draft và tích chọn vào Turn this post into a static page.

    5. Thêm đoạn code html dưới đây vào nội dung của trang:
    <form action="/mail" method="POST">
    <p>Name: <input type="text" name="name" /></p>  
    <p>Email: <input type="text" name="email" /></p>  
    <p>Subject: <input type="text" name="subject" /></p>  
    <p>Content:</p>  
    <p><textarea name="text" cols="40" rows="5"></textarea></p>
    <button type="submit">Submit</button>
</form>

    Lưu ý rằng, chúng ta sẽ sử dụng đường dẫn /mail trong action là địa chỉ xử lý contact form. Bạn có thể thay đổi nó nếu cần.

    1. Publish trang này.

    Tạo trang contact bằng template page-contact.hbs

    1. Tạo mới file page-contact.hbs trong thư mục theme gốc mà bạn đang sử dụng.
    2. Copy nội dung từ file page.hbs (trong theme casper) sang. Nội dung của trang lúc này sẽ như thế này:

    Thêm trang contact vào navigation menu

    Bạn truy câp vào tab Navigation của mục Settings trong phần backend để thêm trang Contact vào thanh điều hướng.

    Cập nhật frontend controller

    Bây giờ, chúng ta cần thêm một phương thức để xử lý gửi mail. Chúng ta sẽ tiến hành cập nhật file core/server/controllers/frontend.js

    Tham chiếu đến Nodemailer

    Thêm mailer = require(‘../mail’) vào phía trên của file:

    var moment      = require('moment'),
    rss         = require('../data/xml/rss'),
    _           = require('lodash'),
    Promise     = require('bluebird'),
    api         = require('../api'),
    config      = require('../config'),
    filters     = require('../filters'),
    template    = require('../helpers/template'),
    errors      = require('../errors'),
    routeMatch  = require('path-match')(),
    mailer      = require('../mail'), // add node mailer

    Thêm phương thức xử lý submit form

    Bạn có thể thêm phương thức này ngay sau rss: function(function (req, res, next){ .. }, và bạn nhớ nhập vào địa chỉ email nhận thư tương ứng nhé.

    //action for posting the contact form
submitContactForm: function (req, res) {

    var mailOptions = {
        // sender address
        from: req.body.email, 

        // reciever email (add your email here)
        to: 'your@email.com', 

        // subject
        subject: req.body.subject, 

        // email body
        html: req.body.text + ' - From: ' + req.body.email 
    };

    //sending the email
    mailer.send(mailOptions).then(function(data) {
        //this is the response to the end user 
        //should ideally redirect or return a view
        res.status(200);
        res.send("Message was successfully sent.");

    }).error(function(error){

        //response for error
        res.status(500);
        res.send('There was an error sending the email.');

    });
}

    Lưu ý rằng, hồi đáp sẽ chỉ trả lại text trên màn hình. Bạn có thể cần chuyển hướng đến một trang khác để đưa người dùng trở về trang chủ nếu cần. Chúng ta sẽ giải quyết vấn đề này ở phần Điều hướng đến một trang tĩnh

    Thêm đường dẫn xử lý form submit

    Chúng ta sẽ tiến hành cập nhật file core/server/routes/frontend.js. Thêm: 

    router.post('/mail', frontend.submitContactForm)

    ngay phía sau đường dẫn: 

    router.get('*', frontend.single);

    Điều hướng đến một trang tĩnh

    Nếu bạn muốn điều hướng đến một trang tĩnh sau khi gửi mail, bạn làm theo các bước sau:

    1. Tạo một trang tĩnh có địa chỉ post url là contact-success và nhập vào nội dung bạn muốn hiển thị khi người dùng gửi mail thành công.
    2. Tạo template page-contact-success.hbs trong thư mục chứa theme.
    3. Cập nhật đoạn code để điều hướng đến trang contact-success bằng lệnh res.redirect(‘/contact-success’). Khi đó, file core/server/controllers/frontend.js trông sẽ như thế này:
    //action for posting the contact form
submitContactForm: function (req, res) {

       ....

    //sending the email
    mailer.send(mailOptions).then(function(data) {

        //this will redirect to the static page you've created
        res.redirect('/contact-success');

    }).error(function(error){

        ....
    });
}

    Sau cùng, nếu bạn muốn trang này tự động điều hướng đến trang chủ sau 5 giây, bạn thêm đoạn code javascript dưới đây vào file page-contact-success.hbs, (ngay sau ):

    <script>

    //redirect to home page after 5 seconds.
    window.setTimeout(function(){

        //set the redirect url here ("/") will redirect to the root
        window.location.href = "/"; 

    }, 5000); //set the time here

</script>

    Thế là xong. Bạn cũng có thể làm tương tự khi gửi mail không thành công.
