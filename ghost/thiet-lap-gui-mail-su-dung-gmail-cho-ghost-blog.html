    Thiết lập gửi mail sử dụng Gmail cho Ghost blog

    08 May 2015 in Ghost  3.73 minutes read

    Để gửi mail, Ghost sử dụng Nodemailer - một module của Node.js cho phép gửi mail từ một ứng dụng Node.js. Nó cho phép chúng ta sử dụng các dịch vụ mail hỗ trợ SMTP phổ biến như Gmail, Mailgun, Yahoo, SES, etc. Trong bài viết này, chúng ta sẽ thiết lập cho Ghost blog gửi mail sử dụng dịch vụ Gmail.

    Mở file config.js bằng trình soạn thảo

    File config.js nằm trong thư mục gốc nơi bạn cài đặt blog. Bạn mở nó với trình soạn thảo mà bạn yêu thích.

    Thêm cấu hình cho Gmail

    Tiếp đến, bạn cần thêm đoạn code dưới đây vào trong file config.js. Bạn thay thế giá trị của user và password cho phù hợp.

    mail: {
    transport: 'SMTP',
    options: {
       service: 'Gmail',
       auth: {
           user: 'your.gmail.id@gmail.com',
           pass: 'your-gmail-password'
      }
    }
}

    Các thiết lập cho mail trong các mode tương ứng trông sẽ như sau:

    Production mode 

    production: {
    url: 'http://my-ghost-blog.com',
    mail: {
        transport: 'SMTP',
        options: {
           service: 'Gmail',
           auth: {
               user: 'your.gmail.id@gmail.com',
               pass: 'your-gmail-password'
          }
        }
    }
}

    Development mode 

    development: {
    url: 'http://my-ghost-blog.com',
    mail: {
        transport: 'SMTP',
        options: {
           service: 'Gmail',
           auth: {
               user: 'your.gmail.id@gmail.com',
               pass: 'your-gmail-password'
          }
        }
    }
}

    Test

    Để kiểm tra các thiết lập gửi mail chúng ta vừa làm có hoạt động hay không, bạn cần Restart Ghost và tiến hành gửi mail phía backend. Trước đó, bạn cần đảm bảo rằng đã thiết lập email cho Ghost User trong trang /ghost/settings/users/.

    Bạn truy cập vào tab Labs trong phần Settings tại địa chỉ /ghost/settings/labs/ và click vào nút Send để gửi email test tới địa chỉ email mà bạn đã thiết lập cho user của bạn.

    Nếu bạn nhận được email tức là các thiết lập đã được thực hiện chính xác.

    Nếu bạn muốn sử dụng dịch vụ gửi mail khác, không phải Gmail chẳng hạn như Mailgun hay Amazon SES thì bạn có thể tham khảo theo hướng dẫn của Ghost Team. Chúc bạn thành công!
